Middleware for logging HTTP requests using Morgan.

**Returns**: function
- Morgan middleware.

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|format|string|✔️|"combined"|Morgan logging format.|
|config|Object|✔️|{}|Additional Morgan configuration options.<br />  See the official morgan package documentation for more details:|

**See**:
- [https://www.npmjs.com/package/morgan](https://www.npmjs.com/package/morgan)

const logger = initLogger("info", transports);
