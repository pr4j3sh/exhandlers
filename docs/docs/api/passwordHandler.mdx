```ts
function passwordHandler(password: string, hashedPassword: string)
```

<div>
  <span className="badge badge--success">async</span>
</div>

Compares a plaintext password with a hashed password.

**Returns**: Promise&lt;boolean&gt;
- `true` if passwords match, otherwise `false`.<br />  See the official bcrypt package documentation for more details:

|PARAMETER|TYPE|OPTIONAL|DEFAULT|DESCRIPTION|
|:---:|:---:|:---:|:---:|:---:|
|password|string|||The plaintext password.|
|hashedPassword|string|||The hashed password for comparison.|

**See**:
- [https://www.npmjs.com/package/bcrypt](https://www.npmjs.com/package/bcrypt)
```js
const isMatch = await passwordHandler("myPassword", storedHashedPassword);
if (isMatch) {
  console.log("Password is correct");
}
```
